    [SMART(no load)/TNT(w/promo)] PLC 5.3 + OpenVPN [Free internet for PC]

    Una sa lahat warning lang 2 sims ko na blocked ang data. Ang bilis mag block ng smart ngayon.
    So sa mga gagamit try this at your own risk.

    UPDATE: (Ongoing experiment) for 936 modems always clear your data history.



    sa ibang device try niyong hanapin yung option para i clear ang data history niyo.

    i dunno if this will work 100% pero nakaka 2GB nako since this morning. Sana hindi ma jinx.

    Para sa mga subscribers ni boss ragsider alam na nila to.
    I made this thread para doon sa nahihirapan parin mag connect gamit ang smart.

    Hindi ko alam kung gagana to sa Luzon

    but im sure this is working for Visayas and Mindanao area.

    What you need?

    1. Smart Sim Preferably Kickstart (kasi ito yung gamit ko)
    2.     PLC 5.3 HTTP INJECTOR
    3.     Latest OpenVPN GUI
    4.Tutorial on how to make your own Payload Configuration.
    5. Proxy Site
    6. VPN account
    http://accountvpn.com/
    https://www.tcpvpn.com/
    https://skyssh.com/
    7. Squid Proxy: http://spys.ru/en/


    FOR TNT you need promo to make it work.
    twitter10 to 4545 works but may capping siya. pero hindi mawawala connection niyo max speed lang if ma cap na is 250kbps. (tested)

    COC10 + T2 (x3) = (not tested)

    you can also try this T10 + T2 (3x) = sabi nila 3 days na net no capping (not tested)
    PS for TNT : m.twitter.com

    same procedure with SMART.


    Procedure:

    1. Run     PLC 5.3 as Admin then load your PAYLOAD
    or follow the     Tutorial on how to make your own Payload Configuration.



    *1. Squid 2. Port 3. Listen Port
    Follow the screenshots below


    *Tips on choosing a Squid proxy
    See screenshots




    mas ok pag port     3128 ang pinili niyo malaki ang chance na mag connect. but not all the time.
    mas mababa ang     latency mas mabilis, mas malaki ang uptime percentage mas stable (i think).



    2. Run Latest     OpenVPN GUI go to C:\Program Files\OpenVPN\config it should look like this



    Paste your     configs and account.txt on this folder

    * Sample Config

    Code:
    http-proxy 127.0.0.1 8080
http-proxy-option VERSION 1.1
route-method exe
client
dev tun
proto tcp
remote 128.200.110.190 443
resolv-retry infinite
nobind
tun-mtu 1500
tun-mtu-extra 32
mssfix 1450
persist-key
persist-tun
auth-user-pass
comp-lzo
reneg-sec 0
verb 3
auth-nocache
    credits to sir mulot

    3. Now Press Play Button on     PLC 5.3 and then Connect to your VPN server.



    NOTE: minsan walang ping kahit buhay yung squid. that means hindi siya compatible sa VPN server mo.
    either humanap ka ng ibang squid or try mong mag connect sa ibang VPN server.

    credits kay boss ragsider dahil sa kanya ko natutunan most of what i shared in this post. yung iba expirements ko nalang.

    Post your feedbacks , questions and critics below ill try to respond to them when i'm not busy.

    Cheers! Enjoy!
